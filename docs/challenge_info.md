## 평가 공식 (★중요★)

오차 (퍼센트):
```
Er_i = 100 × (ActRUL_i - PredRUL_i) / ActRUL_i
```

정확도 점수:
```
         { exp(-ln(0.5) × Er_i/20),  if Er_i ≤ 0  (늦은 예측)
A_RUL = { 
         { exp(+ln(0.5) × Er_i/50),  if Er_i > 0  (이른 예측)
```

최종 점수 = 모든 베어링 A_RUL의 평균

### 핵심: Er은 퍼센트(%)
- 늦게 20% 틀림 = Score 0.5
- 이르게 50% 틀림 = Score 0.5
- 비대칭 비율: 늦은 예측이 2.5배 가혹
